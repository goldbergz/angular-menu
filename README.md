# Angular Menu Application

Небольшое Angular-приложение с меню, чекбоксами и глобальным заголовком.  

## Функциональность

- Две страницы:  
  1. **Home** — приветственная страница с кнопкой перехода на меню.  
  2. **Menu** — список пунктов с чекбоксами.  

- **Выбор пунктов меню**:  
  - Чекбоксы сохраняют выбранные элементы.  
  - Выбранные элементы подсвечиваются и имеют акцентную полоску.  

- **Глобальный заголовок** (`HeaderComponent`):  
  - Отображает количество выбранных пунктов.  
  - Отображает общую сумму числовых значений.  
  - Список выбранных пунктов.

- Состояние меню хранится через **`MenuStateService`** и доступно на всех страницах.


## Установка и запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить приложение:

```bash
ng serve
```


## Технологии

- Angular 17+ (standalone components)
- TypeScript
- SCSS
- Angular Signals для состояния меню
